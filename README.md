# medicover-bot

## Usage

Add `credentials.conf` file to resources with content:

```properties
user: <YOUR_MEDICOVER_ID>
pass: <YOUR_MEDICOVER_PASS>
```

Run from Main.

All features are pending.

## Development

If using IntelliJ, remember about ZIO for IntelliJ plugin.
